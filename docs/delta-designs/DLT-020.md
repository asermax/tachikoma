## Shape

| Part | Mechanism | Flag |
|------|-----------|:----:|
| **S1** | `git_hook` function in `git.py`: bootstrap hook that checks for `.git`, calls `Repo.init()` if missing, configures a fixed committer identity, creates an initial empty commit ("Initialize workspace"), skips if repo already exists | |
| **S2** | `GitRepo` class in `git.py`: wraps a gitpython `Repo` instance at the workspace path. Provides `commit(files, message)` that validates paths, stages specified files, and commits. Returns a structured result (success with commit info, or failure with reason) instead of raising | |
| **S3** | `__main__.py` hook registration: registers git hook after workspace hook | |
| **S4** | `gitpython` added as project dependency in pyproject.toml | |
