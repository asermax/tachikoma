## Shape

| Part | Mechanism | Flag |
|------|-----------|:----:|
| **S1** | `SkillRegistry.refresh()` method that checks a dirty flag — if dirty, saves old dict references, assigns fresh empty dicts, calls `_discover()`, resets the flag, and restores old references on failure; if not dirty, no-op | |
| **S2** | Filesystem watcher using `watchfiles.awatch()` running as a background asyncio task, monitoring `workspace/skills/` recursively for additions, modifications, and deletions | |
| **S3** | 5-second debounce on the watcher — `watchfiles` supports a `debounce` parameter that coalesces burst events before yielding | |
| **S4** | When the watcher yields changes: mark registry dirty + dispatch `SkillsChanged` event (extends `BaseEvent[None]`) on the bubus event bus | |
| **S5** | `SkillsContextProvider.provide()` calls `registry.refresh()` at the start, before reading `registry.skills` | |
| **S6** | Watcher started as a background asyncio task during skills bootstrap, cancelled on graceful shutdown | |
