## Shape

| Part | Mechanism | Flag |
|------|-----------|:----:|
| **S1** | Extend `detect_boundary()` to accept `summary` as `str | None` — when `None`, use an alternative user prompt that omits current-topic framing and frames the task as pure candidate matching | |
| **S2** | Add startup matching path in coordinator's `send_message()` — when no active session exists and candidates are available, run `detect_boundary(summary=None, candidates=...)` before creating a session; on match, reopen directly (no previous session to close); on no-match or error, create fresh session | |
| **S3** | Harden `get_recent_closed()` candidate filtering — verify existing filters (no SDK ID, missing transcript, stale) are comprehensive; add any missing checks | |
| **S4** | Reuse `_persist_bridging_context()` for startup resume — when reopening a session at startup, inject bridging context the same way normal transitions do | |
