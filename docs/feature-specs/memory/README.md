# Memory

## Overview

Automatic memory extraction from conversations. After a conversation ends, the system analyzes the exchange and persists learnings as structured markdown files — episodic summaries, user facts, and preferences — enabling future sessions to reference past interactions.

## Sub-Capabilities

| Capability | Description | Status |
|------------|-------------|--------|
| [memory-extraction](memory-extraction.md) | Post-conversation analysis that extracts and persists memories | ✓ |
