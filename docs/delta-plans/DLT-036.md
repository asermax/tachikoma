#### Step 6: Implement `_idle_close_loop()` method

- Modify: `src/tachikoma/coordinator.py`
- What to do:
  ```python
  # After close_idle_session(), add _idle_close_loop():
  #
  # async def _idle_close_loop(self) -> None:
  #     """Periodic check for idle session timeout."""
  #     while True:
  #         await asyncio.sleep(60)  # 60-second check interval
  #
  #         # Guard: no active session → skip
  #         # Guard: no _last_message_time → skip
  #         # Check: elapsed > _idle_timeout
  #
  #         # Snooze loop if busy:
  #         #   while self._is_busy:
  #         #       snooze = min(300, self._idle_timeout)
  #         #       await asyncio.sleep(snooze)
  #         #       IMPORTANT: After snooze, re-check ALL conditions:
  #         #       - Still have an active session?
  #         #       - Has _last_message_time been reset? (elapsed < timeout)
  #         #         → If yes: break out of snooze loop, return to main
  #         #           60s cycle (timer was reset by new message)
  #         #       - Still busy?
  #         #         → If yes: repeat snooze
  #         #
  #         # If all checks pass: await self.close_idle_session()
  #
  # Outer try/except catches per-iteration errors (logged, loop continues)
  # CancelledError propagates cleanly for shutdown
  ```
- Verify: `just lint && just typecheck`
