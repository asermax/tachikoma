## Notes

- **Decision for reconciliation (DES candidate)**: Always auto-generate migrations from model changes using Alembic autogenerate, rather than hand-writing revision scripts. The only exception is the initial baseline revision which captures the pre-Alembic schema. This ensures migrations accurately reflect model state and avoids drift between ORM models and migration scripts.
